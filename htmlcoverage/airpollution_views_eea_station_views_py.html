54 """ 

55 Return surface pollution readings from base stations.

57 :param start_date: YYYYMMDD. Start date of the observations to return.

59 :param time_type: 'hourly' | 'day_mean' | 'rolling'. 

60 'hourly' - all available hours for the day 

61 'day_mean' - a mean value for all available and valid hours for each day 

62 'rolling' - An 8-hour rolling mean for the day. Each hour value provided is the mean of that hour and the previous 7. 

63 :param country_code: Optional. 2-digit country code. If country_code and nuts_region are supplied, the nuts_region will take precidence. 

64 :param nuts_region: Optional. Nuts region. All regions are supported. If country_code is also supplied, the NUTS region will take precidence. 

65 :param pollutant: Optional. pollutant code of pollutant. If none, all are returned. 

66 :return: JSON dictionary of the results. 

67 """
